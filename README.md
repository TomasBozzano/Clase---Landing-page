# Sitio web inspirado en Junín, Buenos Aires

Este proyecto es un homenaje a Junín, mi ciudad natal, y tiene como propósito principal servir como una plataforma de aprendizaje para mejorar mis habilidades en HTML, CSS y JavaScript.

El sitio busca destacar los encantos de Junín, Provincia de Buenos Aires, mostrando sus lugares emblemáticos, historia y cultura. Además, incluye videos de YouTube para ofrecer una experiencia inmersiva y permitir que los visitantes aprecien los diversos lugares que el partido de Junín tiene para ofrecer.

## Características principales
- **Diseño responsivo:** El sitio se adapta a diferentes dispositivos para asegurar una buena experiencia de usuario.
- **Contenido multimedia:** Videos integrados de YouTube que muestran los lugares destacados de Junín.
- **Historia de Junín:** Información sobre la fundación y datos relevantes del municipio.
- **Galería visual:** Imágenes que reflejan la belleza y singularidad de la ciudad.

## Tecnologías utilizadas
- **HTML5:** Para la estructura del sitio.
- **CSS3:** Para el diseño y la estética.
- **JavaScript:** Para agregar interactividad.

## Objetivos del proyecto
1. Aprender y mejorar el uso de HTML, CSS y JavaScript.
2. Crear un espacio que represente visualmente a Junín y comparta su historia.
3. Explorar la integración de videos de YouTube en un sitio web.

## Cómo usar este proyecto
1. Clona este repositorio:
   ```bash
   git clone https://github.com/TomasBozzano/Landing-page-Junin.git
   ```
2. Abre el archivo `index.html` en tu navegador para visualizar el sitio.
